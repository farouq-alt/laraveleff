🔄 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════

User Views Seminars List:
  User → /seminaires → Route → Controller → Model → Database
                                  ↓
                                View → HTML → Browser

User Views Seminar Details:
  User → /seminaires/1 → Route → Controller → Model → Database
                                      ↓
                                    View → HTML → Browser

User Deletes Seminar:
  User → Click Delete → Confirm → DELETE /seminaires/1
                                      ↓
                                   Controller → Model → Database
                                      ↓
                                   Redirect → List → Browser
